# 🧮 Calculadora Simples (HTML, CSS e JavaScript)

Projeto básico de uma calculadora web desenvolvida com **HTML**, **CSS** e **JavaScript**, ideal para iniciantes que estão aprendendo lógica de programação e manipulação do DOM.

---

## 🚀 Funcionalidades

- ✅ Soma
- ✅ Subtração
- ✅ Multiplicação
- ✅ Divisão
- ✅ Validação de campos vazios
- ✅ Tratamento de divisão por zero
- ✅ Interface simples e responsiva

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
